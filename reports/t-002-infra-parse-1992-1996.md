## Summary

Extracted metadata from `1992-1996_Honda_Prelude_H22A1_Helms_Manual.pdf`. The file was discovered to be an HTML download page from club-honda.eu (not a PDF), so a robust file-type detection mechanism was built into the parser. Key findings: the actual manual is 38.2 MB, contains 1,408 pages, covers Honda Prelude IV BA8/9, BB1-BB4 (1992–1996), and includes 23 sections with standard service manual structure. A reusable Python script (`scripts/extract_pdf_metadata.py`) was created that handles both real PDFs (via `pdfinfo`/`pypdf`) and HTML download pages, and is designed for reuse by T-003 (the BB6 manual).
