## Summary

Extracted comprehensive metadata and table of contents from `1997-2001_Honda_Prelude_BB6_Helms_Manual.pdf` (170 MB). The PDF is entirely image-based (scanned pages created with Adobe Acrobat 6.0 in 2005), so text extraction returns empty — bookmarks/outline entries were extracted via pypdf instead. Results were appended to `research/indexes/manual-metadata.md` alongside the existing T-002 section. OCR cross-reference revealed a page count discrepancy: PDF has 1,376 pages but bb6_ocr/images/ contains 1,411 PNG files (35 extra "p"-prefixed files + multi-scan duplicates).
