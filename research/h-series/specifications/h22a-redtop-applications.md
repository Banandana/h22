## Summary

The Honda H22A Redtop (220 PS) was fitted to three JDM vehicle applications across two generations of chassis codes: the 4th-generation Prelude (BB1/BB4/BB6/BB8), the 5th-generation Prelude (BB6/BB8), and the 6th-generation Accord/Torneo Euro-R (CL1). It was exclusive to top-trim SiR S-Spec, Type S, and Euro R models — never offered in base or mid-tier trims. The Redtop's applications spanned from 1992 through 2001, with a notable gap between the 4th-gen Prelude (1992–1996) and the Accord Euro R (1999–2001), during which the Redtop lived on exclusively in the 5th-gen Type S. All Redtop applications used a closed-deck block (1992–1996) or open-deck block (1997–2001), with the engine code stamped on the block serial number prefix (H22A-4xxxxxx for Type S/Euro-R).

## Market Exclusivity

The H22A Redtop was **never officially sold outside Japan**. No USDM, EDM, or AUS model received the 220 PS Redtop variant. This exclusivity has significant implications:

| Market | Highest Output H22 Variant | Power | Notes |
|--------|---------------------------|-------|-------|
| JDM | H22A Redtop | 220 PS | Prelude Type S, SiR S-Spec, Accord Euro R |
| USDM | H22A4 | 200 PS (195 hp) | Prelude Base, Type SH (5th gen); 190 PS (H22A1) in 4th gen |
| EDM | H22A8 | 200 PS | Prelude VTi-S (1999–2001); H22A7 at 212 PS in Accord Type R |
| AUS | H22Z1 | 200–203 PS (disputed) | Prelude VTi-R; speculation of increased output |

**Import pathways:** Redtop-equipped vehicles entered non-JDM markets through:
1. **Official export programs** — limited numbers of Type S Preludes were exported to select markets (e.g., Australia, Middle East) as gray-market imports
2. **Parts cars** — engines and components sourced from Japanese wrecks
3. **Private import** — individuals importing complete vehicles under show-or-store exemptions (US) or similar regulations

## Notes

### Key Observations

1. **The "Type S" nameplate carried the Redtop.** In JDM Honda nomenclature, "Type S" denoted the highest-performance trim level. The 5th-gen Type S was the only mass-market Honda coupe to feature the 220 PS Redtop as a factory option.

2. **The Accord Euro R was a sedan, not a coupe.** Despite sharing the Redtop with the Prelude Type S, the Euro R was a 4-door sedan (and its hatchback twin, the Torneo). This made it a practical daily driver with supercar-level power output — an unusual combination in the late 1990s.

3. **Serial codes are the definitive identifier.** Visual inspection of the valve cover color is unreliable (aftermarket covers exist, repainting occurs). The engine block serial number is the only reliable way to confirm a Redtop engine.

4. **The CD6 Accord SiR conflict.** Some sources list the CD6 Accord SiR (1994–1997) as receiving the 220 PS Redtop, while others list it at 190 PS (Blacktop). This may reflect different model years within the CD6 production run — earlier units may have been 190 PS Blacktop, later units upgraded to 220 PS Redtop. Further verification needed against Honda service bulletins.

### Gaps Identified

- **Exact production numbers per application** (how many Type S vs. SiR S-Spec vs. Euro R) are not published by Honda.
- **Export program documentation** for non-JDM Redtop imports is fragmented and incomplete.
- **Dealer brochure specifications** from 1992–2001 are not readily accessible online.
- **Engine swap compatibility matrix** between Redtop applications (e.g., can a Type S engine directly swap into an Accord Euro R?) is not fully documented.
