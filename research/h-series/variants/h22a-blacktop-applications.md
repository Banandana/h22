## Summary

The H22A Blacktop was installed in four distinct JDM application families: the 4th generation Prelude SiR (BB1/BB4, 1992–1996), the 5th generation Prelude SiR (BB6/BB8, 1997–2001), the Accord SiR sedan/wagon (CD6/CF2, 1994–1997), and the rare Accord SiR Coupe (CD8, 1994, NZ export). Contrary to what the task description implies, the Torneo Euro-R (CL1, 1999–2001) did NOT receive a Blacktop variant — it was equipped exclusively with the H22A Redtop (220 PS). Internationally, detuned Blacktop variants (H22A1 USDM, H22A2/H22A5 EDM, H22Z1 AUS) shared the same basic architecture with market-specific calibrations and emissions equipment.

## Torneo — Important Clarification

**The Torneo Euro-R (CL1, 1999–2001) did NOT receive the H22A Blacktop.** It was equipped exclusively with the **H22A Redtop** (220 PS @ 7,200 RPM, 221 Nm @ 6,500 RPM, 11.0:1 CR).

This is a common point of confusion because:
1. The Torneo was positioned as a "sports sedan" similar to the Accord SiR
2. The Accord SiR received the Blacktop, leading to assumptions about the Torneo
3. The Torneo Euro-R's name ("Euro R") parallels the Accord Euro R, which also received the Redtop

**Torneo Euro-R specifications:**
- Engine: H22A Redtop (220 PS @ 7,200 RPM)
- Chassis: CL1
- Production: 1999–2001
- Features: Recaro seats, MOMO steering wheel, Torsen LSD, sport suspension, 4-2-1 stainless headers, HID headlights
- Weight: 1,330 kg
- 0-100 km/h: ~6.5 seconds

**Other Torneo trims** (F18B, F20A, F20B engines) did not receive any H22A variant.

## Notes

### Conflicts and Disputes

1. **Accord SiR power output:** Multiple authoritative sources list the CD6 Accord SiR at 190 PS (Blacktop), while some sources (mywikimotors) list it at 220 PS (Redtop). Working hypothesis: production-year transition from Blacktop to Redtop during the CD6 run (early models 1994–1995 = Blacktop, later models 1996–1997 = Redtop). The block serial code ranges (H22A-2000001~ through H22A-2070001~) span both possibilities. **Requires verification against Honda service bulletins.**

2. **Torneo Euro-R engine:** The task description mentions "Torneo" in the context of Blacktop applications. However, all authoritative sources (Wikipedia, Honda Thomsen, 8020automotive, Goo-Net Exchange, CarBuzz) confirm the Torneo Euro-R received the **Redtop** (220 PS), not the Blacktop. This is a common misconception. The Torneo did not receive any Blacktop variant.

3. **H22Z1 power output:** Australian H22Z1 is listed at 200 PS by most sources, but the Prelude owner manual lists a compression ratio of 11.0:1 (same as Redtop), leading to speculation of 203 PS output. **Unresolved — requires access to Australian Honda specification sheets.**

4. **VTEC engagement RPM:** Sources vary between 5,200 RPM and 5,800 RPM. The variation likely reflects different calibration between 4th gen and 5th gen ECUs. **Needs further verification.**

### Identification Tips

- **Valve cover color** is the primary visual identifier: black = Blacktop, red = Redtop (but covers are frequently repainted)
- **Block serial codes** provide definitive identification (see h22a-blacktop-jdm-origin.md for complete serial code reference)
- **Throttle body size:** 60 mm = Blacktop, 62.5 mm = Redtop
- **Exhaust diameter:** 51 mm = Blacktop, 57 mm = Redtop
- **Camshaft duration:** 288°/308° = Blacktop, 306°/310° = Redtop
- **Intake manifold:** Single runner = Blacktop (most variants), Dual runner = Redtop
- **Note:** Valve covers are frequently repainted — always verify with block serial code or internal components
