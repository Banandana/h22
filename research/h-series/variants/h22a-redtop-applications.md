## Summary

The Honda H22A Redtop engine (220 PS, 11.0:1 compression, red valve cover) was installed in four distinct JDM vehicle applications across the 1992–2001 production run: the 4th-gen Prelude SiR S-Spec (BB1/BB4, 1992–1996), the 5th-gen Prelude Type S (BB6, 1997–2001), the Accord SiR sedan/wagon (CD6/CF2, 1994–1997), and the Accord Euro-R / Torneo Euro-R (CL1, 1999–2001). Each application received a unique calibration, intake manifold design, and feature package. The Prelude SiR S-Spec emphasized track-oriented luxury (Alcantara, Recaro, ATTS); the Type S added Dynamic Chambering airbox and Active ABS; the Accord SiR shared the same Redtop powertrain but in a sedan/wagon body; and the Euro-R featured a single-runner intake manifold tuned for different power delivery characteristics. All four applications were JDM-exclusive — none were sold in North America or Europe.

## Notes

### Key Observations

1. **Four distinct applications, one engine.** The H22A Redtop appeared in exactly four JDM vehicle applications: Prelude SiR S-Spec (BB1/BB4), Prelude Type S (BB6), Accord SiR (CD6/CF2), and Accord Euro-R/Torneo Euro-R (CL1). All were JDM-exclusive.

2. **Intake manifold is the key differentiator.** The Prelude applications used a **dual-runner** intake manifold, while the Accord Euro-R used a **single-runner** manifold. This changes the power delivery character — dual-runner favors high-RPM peak power, single-runner favors broader mid-range torque.

3. **The Accord SiR power output is disputed.** Sources conflict between 190 PS (Blacktop) and 220 PS (Redtop) for the CD6/CF2 Accord SiR. The most likely explanation is a production-year transition: early models (1994–1995) received the Blacktop, while later models (1996–1997) received the Redtop. This needs verification against Honda service bulletins.

4. **ATTS was a Type S exclusive.** The Active Torque Transfer System (electronically controlled clutch pack differential) was standard on the Type S and available on select SiR S-Spec models, but not on the Accord SiR or Euro-R.

5. **The Euro-R was the last Redtop application.** The CL1 Accord Euro-R (1999–2001) was the final vehicle to receive the H22A Redtop before it was replaced by the K20A in subsequent Honda performance sedans.

6. **Dynamic Chambering was a Type S innovation.** The variable-length intake airbox on the 5th-gen Type S was a new technology for the H22A, replacing the fixed-volume plenum of the 4th-gen. It optimized plenum volume for both low-end torque and high-RPM power.

7. **All Redtop applications were JDM-exclusive.** Despite the Prelude's popularity in North America, the Redtop-powered versions were never sold in the USDM or EDM markets. USDM Preludes received the detuned H22A1 (190 PS, Blacktop) or H22A4 (200 PS, Blacktop).

### Gaps Identified

- **Accord SiR (CD6/CF2) exact power output per model year** — no definitive source confirms which years received Redtop vs Blacktop. Honda service bulletins or factory specification sheets would resolve this.
- **Block serial code to engine type mapping for CD6/CF2** — the serial code ranges (H22A-2000001~ through H22A-2070001~) are listed but not correlated to Redtop vs Blacktop.
- **JC08 fuel economy figures for Prelude SiR S-Spec and Type S** — only the Euro-R has published JC08 figures (11.6 km/L).
- **Exact curb weights for 4th-gen SiR S-Spec variants** — weights are estimated from 5th-gen data and general 4th-gen specs.
- **Factory maintenance schedules specific to Redtop applications** — general H-series intervals are documented but Redtop-specific intervals (especially for the 11.0:1 CR) are not separately specified.

## Conflicts and Ambiguities

1. **Accord SiR (CD6/CF2) power output:** Multiple authoritative sources list the CD6 Accord SiR and CF2 Accord SiR Wagon with 190 PS (Blacktop, 10.6:1 CR), while other sources (mywikimotors) list them with 220 PS (Redtop, 11.0:1 CR). The most likely explanation is a production-year transition — early CD6 models (1994–1995) received the Blacktop, and later models (1996–1997) received the Redtop. The block serial code ranges (H22A-2000001~ through H22A-2070001~) span both possibilities. **Working hypothesis:** 190 PS for 1994–1995, 220 PS for 1996–1997. Requires Honda service bulletin verification.

2. **VTEC engagement RPM for Euro-R:** The single-runner intake manifold on the Euro-R may shift VTEC engagement compared to the dual-runner Prelude. No source explicitly documents Euro-R-specific VTEC engagement parameters. Using the general Redtop figure of 5,200–5,500 RPM as a working estimate.

3. **Accord SiR Wagon (CF2) production duration:** One source states 1997 only; another implies 1994–1997. The wagon variant was likely produced for a shorter period than the sedan. Further verification needed.

4. **4th-gen SiR S-Spec weight:** No source provides an exact curb weight for the 4th-gen SiR S-Spec. Estimates derived from 5th-gen Type S weight (1,310 kg) adjusted for 4th-gen body differences (~1,290–1,310 kg).
