## Summary

The Honda H22A Redtop engine was paired exclusively with close-ratio 5-speed manual transmissions across all JDM applications (Prelude SiR S-Spec, Prelude Type S, Accord SiR, Accord Euro-R). The primary factory transmission for JDM SiR/S-Spec models was the M2B4 (with viscous LSD), while non-SiR variants used the M2A4 (open diff). Both share identical gear ratios with a 4.266:1 final drive. The USDM H22A1 received the M2F4, which has slightly taller 4th and 5th gears but the same final drive. All H22A transmissions use a viscous-coupling limited-slip differential when equipped. The transmissions are fully interchangeable with other H-series and F-series transmissions (M-prefix), though ATTS-equipped units require modification to fit non-ATTS chassis.

## Limited-Slip Differential (LSD)

### Factory LSD Type: Viscous Coupling

All factory H-series transmissions with LSD use a **viscous coupling** limited-slip differential. This is confirmed by community consensus across multiple forums:

| Transmission | LSD Type | Locking Characteristic |
|--------------|----------|----------------------|
| M2B4 (JDM SiR) | Viscous LSD | Progressive lock-up based on temperature/slip |
| M2T4 (5G SiR LSD) | Viscous LSD | Same viscous type |
| M2Z4 (5G SiR S-Spec) | Viscous LSD | Same viscous type |
| T2T4 (Accord SiR-T) | Helical LSD | Pre-load type, smoother operation |
| T2W4 (Accord Euro-R) | Helical LSD | Pre-load type, smoother operation |
| U2Q7 (EDM Accord Type-R) | Helical LSD | Pre-load type, smoother operation |
| M2U4 (Type-SH ATTS) | ATTS system | Electronically controlled clutch pack (NOT a traditional LSD) |

**Important:** The M2B4 (the primary LSD transmission for 4th-gen SiR) uses a **viscous LSD**, which provides progressive, temperature-dependent lockup. This is less aggressive than the helical LSD found in later Accord Euro-R (T2W4) and EDM Type-R (U2Q7) transmissions. For track use, many enthusiasts upgrade to a helical LSD (Quaife or MFactory) or swap to a T2W4/T2T4 transmission.

### Aftermarket LSD Options for M2B4/M2A4

## Final Drive Ratio

All H22A transmissions use a **4.266:1 final drive** ratio. This is consistent across:

- M2A4 (JDM STD)
- M2B4 (JDM LSD)
- M2F4 (USDM)
- M2U4 (Type-SH ATTS)
- T2T4 (Accord SiR-T)
- T2W4 (Accord Euro-R)
- U2Q7 (EDM Type-R)
- M2Z4 (5G SiR S-Spec)

**Aftermarket final drive options:** MFactory offers a 4.64:1 final drive set for H-series transmissions, providing approximately 8.5% more effective gear reduction for track/drag applications.

## Transmission Identification

All H-series transmissions can be identified by the code stamped on the bellhousing or on a metal tag attached to the transmission case. The code follows the pattern **MXX#** where:

- **M** = Honda Prelude (M-prefix denotes Prelude transmissions)
- **XX** = Variant identifier (A, B, F, etc.)
- **#** = Generation/batch number (4, 5, etc.)

To identify an unknown transmission:
1. Look for a metal tag or stamped code on the bellhousing
2. Count gear teeth on input/output/countershafts (see Honda-Tech FAQ "How to determine tranny gear ratios without counting teeth")
3. Compare against the master list in this document

All H-series transmissions (M-prefix) are **fully interchangeable** with each other in terms of bellhousing pattern, clutch, flywheel, and driveshaft mounting. The only exception is ATTS transmissions (M2U4, T2W4 with ATTS hardware), which require removal of the ATTS mechanism to fit non-ATTS chassis.

## Common Issues

### 5th Gear Synchro Wear

A well-documented issue with all H-series transmissions is premature 5th gear synchro wear, particularly in the M2B4 and M2F4. Symptoms include grinding when shifting into 5th under load. Honda issued a Technical Service Bulletin (TSB) for this issue, but reports indicate the fix is often temporary. Affected components:

- 5th/R synchronizer sleeve assembly
- 5th gear blocker ring
- Shift linkage bushings (contributes to poor engagement)

### Oil Specification

H-series transmissions use **manual transmission oil** (not engine oil). Recommended specification:

| Parameter | Value |
|-----------|-------|
| Viscosity | 75W-90 GL-4 |
| Capacity | ~2.1 liters (drain and refill) |
| OEM recommendation | Honda Manual Transmission Fluid (part 08200-9008) |
| Aftermarket equivalent | Motul 75W-90, Red Line MT-90, AMSOIL Synthetic Manual Transmission Fluid |

**Warning:** Do not use GL-5 oil in H-series transmissions — the sulfur-containing additives corrode the brass synchro rings. Only GL-4 or specifically rated synthetic manual transmission fluids should be used.

## Notes

- **Conflicting data on M2A4 3rd gear:** One source (SuperHonda forum) lists M2A4 3rd gear as 1.160, while all other sources (PreludeOnline, Honda-Tech master list by Kronn and Bense, NZHondas service manual reference) list it as 1.360 — identical to the M2B4. The weight of evidence supports 1.360; the 1.160 figure may be a typo or confusion with another transmission.
- **M2F4 5th gear discrepancy:** Some sources list M2F4 5th as 0.871 rather than 0.870. This is a rounding difference and does not affect practical performance.
- **Viscous LSD maintenance:** Viscous LSD units do not require regular fluid changes, but the housing should be inspected for leaks. If the LSD becomes sluggish (common after 15+ years), replacement with a helical LSD or new viscous unit is recommended.
- **ATTS transmissions cannot be used** in non-ATTS vehicles without significant modification. The ATTS mechanism occupies space within the differential housing and requires additional wiring, ECU integration, and hydraulic lines.
