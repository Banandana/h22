## Summary

The Honda H22A4 was the sole engine offered in the fifth-generation Honda Prelude (BB6 chassis, 1997–2001). It powered three North American trim levels — Base (USDM/CAN), SE (Canada exclusive), and Type SH (USDM/CAN) — plus Australian VTi-R/VTi-R ATTS (1997–1998 with H22A4 code; switched to H22Z1 in 1999–2001) and European 2.2 VTi/2.2 VTi-S (which used H22A5/H22A8, mechanically nearly identical). The USDM lineup was dramatically simplified compared to previous generations, offering only two trims (Base and Type SH) with a single powertrain. The Canadian market added the SE as a mid-tier option in 2001. Key model year changes include a 5 hp power bump for manual transmission models from 195 to 200 hp at the 1999 facelift, and the introduction of Sequential SportShift on automatics. The Type SH featured Honda's world-first production Active Torque Transfer System (ATTS) torque-vectoring technology. All fifth-gen Preludes shared the BB6 chassis code for 2WS variants, with BB8 for 4WS versions sold in foreign markets.

## Fifth Generation Prelude Overview (BB6 Chassis)

### Design Context

The fifth-generation Prelude (chassis codes BB5–BB9) arrived for the 1997 model year, produced from November 1996 through October 2001. Designer Manabu Konaka led the project starting in 1993. The styling represented a deliberate return to the angular, squared-off proportions of the first three generations of Prelude, moving away from the softer, more rounded "liquefied" sheet metal of the controversial fourth generation (1992–1996). Fixed headlights replaced pop-ups, and the overall silhouette balanced retro cues with late-1990s aerodynamic refinement.

The BB6 chassis was 1,780 mm long, 1,750 mm wide, 1,315 mm tall, with a 2,585 mm wheelbase — slightly longer than the 4th-gen BB1 (approximately 1,760 mm long). Curb weight ranged from 1,340 kg (2,954 lb) for the Base model to 1,380 kg (3,042 lb) for the Type SH. Body stiffening measures over the predecessor included a reinforced rear bulkhead, additional crossmembers, and a steering-column support stretching between the A-pillars.

All fifth-gen Preludes stayed within the BB-chassis code family:

**Key point for H22A4:** The H22A4 engine appears exclusively under chassis code **BB6** for 2WS applications. The H22A4-powered 4WS variants would be coded **BB8**, though these were not sold in North America (4WS was not offered in USDM/CAN for the 5th gen).

## USDM Applications (United States)

### Model Lineup

The USDM lineup was dramatically simplified compared to previous generations. Only **two trims** were offered: **Base** and **Type SH**. Both used the H22A4 engine. This was a strategic cost-cutting measure — Honda consolidated the entire USDM lineup under a single powertrain.

| Feature | Base | Type SH |
|---------|------|---------|
| **Engine** | H22A4 2.2L DOHC VTEC | H22A4 2.2L DOHC VTEC |
| **Power (1997–1999)** | 195 hp @ 7,000 RPM | 195 hp @ 7,000 RPM |
| **Power (1999–2001 M/T)** | 200 hp @ 7,000 RPM | 200 hp @ 7,000 RPM |
| **Power (1999–2001 A/T)** | 195 hp @ 7,000 RPM | N/A (MT only) |
| **Torque** | 156 lb-ft @ 5,250 RPM | 156 lb-ft @ 5,250 RPM |
| **Transmission** | 5-speed MT or 4-speed AT (SportShift) | 5-speed MT only |
| **Chassis Code** | BB6 | BB6 |
| **Steering** | Variable-assist rack & pinion | Variable-assist rack & pinion (15.61:1 ratio, slightly quicker) |
| **Curb Weight** | ~1,340 kg (2,954 lb) | ~1,380 kg (3,042 lb) |

The Type SH added approximately $2,500–$2,770 to the base price, primarily for the ATTS system (~$1,500 option value) and sportier wheels/spoiler.

### Base Trim — Standard Equipment

The Base trim was well-equipped for a sports coupe of its era:

| Category | Equipment |
|----------|-----------|
| **Exterior** | 16×6.5 in cast aluminum alloy wheels (5-spoke directional), 205/50VR16 87V tires, body-color bumpers/handles/mirrors, rear spoiler, fog lights, steel sliding sunroof |
| **Interior** | Cloth upholstery, driver seat height adjustment, tilt steering wheel, cruise control, dual front airbags, remote keyless entry (1999+), 6-speaker stereo with cassette player, trip computer |
| **Safety** | 4-wheel ABS (standard), dual front airbags, 3-point seatbelts all positions, side-impact door beams |
| **Mechanical** | Double wishbone suspension (F/R), 11.1-in vented front disc brakes, 10.25-in rear disc brakes, 51 mm exhaust, H22A4 engine |

### Type SH Trim — Additional Equipment Over Base

The Type SH ("Super Handling") was the flagship USDM trim, featuring Honda's world-first production Active Torque Transfer System (ATTS):

| Category | Additional Equipment vs. Base |
|----------|------------------------------|
| **ATTS System** | Electromechanical torque-vectoring differential — sends up to 80% of available torque to outside front wheel during cornering; adds ~44 lbs; uses hydraulic clutch pack + triple-pinion planetary gears + MCU-controlled solenoid valves |
| **Wheels** | Unique Type SH 5-spoke alloy wheels (different design from Base), Goodyear Eagle Sport All-Season 205/50R16 87V M+S tires |
| **Exterior** | Rear spoiler (standard on SH), Type SH badging |
| **Interior** | Leather-wrapped shift knob, perforated leather seats (some sources), simulated carbon fiber trim on door panels and audio panel |
| **Steering** | Slightly quicker ratio (15.61:1 vs 15.6:1 on Base) |
| **Transmission** | 5-speed MT only (M2U4 code); gear ratios identical to Base M2Y4 |

**Important note:** Most Base Preludes had automatic transmissions, while SH versions were 5MT only. The ATTS system was rarely chosen — few Type SHs left the factory, making them the rarest and most sought-after 5th-gen Preludes.

### Automatic Transmission — Sequential SportShift

Both Base trims (not Type SH, which was MT-only) offered a 4-speed automatic with Honda's new **Sequential SportShift** feature:

| Feature | Details |
|---------|---------|
| **Gear Ratios** | 1st: 2.528, 2nd: 1.427, 3rd: 0.976, 4th: 0.653, Reverse: 1.863, Final Drive: 4.785 |
| **SportShift Operation** | Gear lever can be left in Drive for automatic shifting, or tipped forward/backward for manual gear selection |
| **Market Reception** | Novel feature but not widely used; many owners reported never engaging manual mode |
| **Power Output (A/T)** | 195 hp @ 7,000 RPM (all years, including 1999+ facelift — A/T did not receive the +5 hp bump) |

### SE Trim — Canada Exclusive (2001)

The SE was introduced in 2001 as a replacement for the Type SH, which was discontinued in Canada. Mechanically identical to the Base model, the SE was a **cosmetic and comfort package**:

| Category | Equipment |
|----------|-----------|
| **Seats** | Perforated, heated leather seats |
| **Wheels** | Type SH Enkei-style rims (same design as SH) |
| **Exterior** | Type SH rear spoiler |
| **Interior** | Leather-wrapped shift knob, simulated carbon fiber trim kit on door panels and audio panel |
| **What SE Did NOT Have** | Body-colored side skirts (Type SH had these as standard), ATTS torque-vectoring system |
| **Mechanical** | Identical to Base — same H22A4, same suspension, same brakes, same steering ratio |

**Canadian-specific features (all trims):**
- Heated seats as standard equipment across all trims
- 4.5-liter washer fluid reservoir (vs. 2.5-liter US spec)
- Dealer accessories available: carbon fiber audio panel, sunroof visor, 6-disc-in-dash CD changer, trunk-mounted CD changer, cassette player, roof rack, gold-plated emblem kit, gold-plated exhaust finisher, leather-wrapped shift knob, full and half nose mask, security system, cargo mat

## Australian Market Applications

### Model Lineup

Australia received the H22A4 under two different engine codes depending on model year:

| 1997–1998 | VTi-R | **H22A4** | 203 PS (200 hp) | BB6 | 2WS |

| 1999–2001 | VTi-R | **H22Z1** | 200–203 PS (disputed) | BB6 | 2WS |
| 1999–2001 | VTi-R ATTS | **H22Z1** | 200–203 PS (disputed) | BB6 | 2WS |

**Weight comparison (Australian models):**
| Trim | Weight (MT) | Weight (AT) | Weight (ATTS) |
|------|-------------|-------------|---------------|
| VTi-R / VTi-R ATTS | 1,268 kg (2,795 lb) | 1,298 kg (2,862 lb) | 1,308 kg (2,884 lb) |

**Note on H22Z1:** From 1999 onward, Australian models switched from H22A4 to H22Z1. The H22Z1 is thought to be mechanically identical to the H22A4 but with disputed power output (200–203 PS). It remains ATTS-compatible.

## European Market Applications

European Preludes used the H22A4's close relatives — the **H22A5** (1997–1998) and **H22A8** (1999–2001) — rather than the H22A4 itself. These differ mainly in ECU calibration, emissions equipment, and minor intake/exhaust routing to satisfy regional standards. Block, head, valvetrain, and internals are shared.

## Japanese Domestic Market Applications

JDM Preludes received a wider range of trims, but **none used the H22A4 engine code**. The H22A4 was specific to USDM/CAN/AUS (1997–1998) and certain export markets. JDM models used:

| Trim | Engine | Power | Chassis Code | Steering |
|------|--------|-------|--------------|----------|
| Xi | F22B 2.2L SOHC | 135 PS | BB5 | 2WS |
| Si | F22B 2.2L SOHC | 160 PS | BB5/BB7 | 2WS/4WS |
| SiR | H22A Blacktop 2.2L DOHC VTEC | 200 PS | BB6/BB8 | 2WS/4WS |
| SiR S-spec | H22A Redtop 2.2L DOHC VTEC | 220 PS | BB6 | 2WS |
| **Type S** | H22A Redtop 2.2L DOHC VTEC | 220 PS | BB6 | 2WS |

**Type S (JDM exclusive):** The highest-performance JDM variant, equipped with the H22A Redtop (220 PS @ 7,200 RPM, 11.0:1 compression), ATTS, Active Control ABS, and unique interior (Cabron/Excene synthetic materials with Alcantara and red stitching). Notably, the Type S did **not** have a sunroof — all other JDM Preludes did.

## Notes

### Conflicts & Uncertainties

1. **SE trim availability:** Multiple sources confirm SE was a Canada-only trim introduced in 2001 as a Type SH replacement. However, some sources (e.g., HotCars) suggest SE existed in earlier years. Resolution: Wikipedia's dedicated "Canadian market" section confirms SE was introduced in 2001 as the Type SH replacement in Canada. The SE was mechanically identical to Base with cosmetic/comfort upgrades.

2. **Automatic transmission power in 1999+:** Car and Driver states the 1999 facelift gave +5 hp to manual models (195→200 hp). Some sources imply automatics also gained power. Resolution: Car and Driver's "By the Numbers" review confirms 200 hp @ 7,000 rpm for the 1999 SH (which was MT-only). For Base models, the automatic remained at 195 hp throughout the generation. The +5 hp bump applied to USDM manuals only.

3. **Australian H22A4 vs H22Z1:** The transition from H22A4 to H22Z1 in Australia occurred around 1999. Some sources list Australian VTi-R as using H22A4 through 2001. Resolution: Wikipedia's model matrix clearly shows H22A4 for 1997–1998 and H22Z1 for 1999–2001 in Australia. The H22Z1 is believed to be mechanically identical to the H22A4 with disputed power output.

4. **Canadian SE vs USDM Base exterior differences:** Some sources suggest the Canadian SE had body-colored side skirts like the Type SH. Resolution: Community consensus on PreludePower confirms the SE did NOT have body-colored side skirts as standard — it lacked the ATTS system entirely and was essentially a cosmetic package over the Base model.

5. **Type SH production numbers:** Few Type SHs were produced compared to Base models. The ATTS system was expensive (~$1,500 option) and rarely chosen. This makes surviving Type SH examples particularly rare and valuable today. Exact production numbers per trim are not publicly available from Honda.

### Key Takeaways

- The H22A4 was the sole USDM engine for the 5th-gen Prelude (BB6, 1997–2001), offered in Base and Type SH trims
- Canada received an additional SE trim (2001 only), mechanically identical to Base with cosmetic upgrades
- Australia received the H22A4 under VTi-R/VTi-R ATTS badging (1997–1998), then switched to H22Z1 (1999–2001)
- Europe received H22A5/H22A8 (mechanically near-identical to H22A4) under 2.2 VTi/2.2 VTi-S badging
- Japan did not use the H22A4 code — instead used H22A (Blacktop/Redtop) and F22B engines
- The 1999 facelift added +5 hp for manual transmission models (195→200 hp) and a new front grille with "Prelude" badge
- Type SH was MT-only and featured ATTS torque-vectoring — the world's first production application
- Automatic transmissions featured Sequential SportShift (manual mode via gear lever)
- All trims shared the same BB6 chassis code for 2WS; 4WS variants used BB8 (not sold in NA)
- The USDM lineup was simplified to just two trims — a cost-cutting measure that reduced consumer choice

### Implications for Building/Modding

- **Base and Type SH share identical mechanicals** except for ATTS hardware and steering ratio — any H22A4 swap will work in either chassis
- **Type SH ATTS can be retrofitted** to Base models with aftermarket conversion kits, but the system is complex and rare
- **1999+ manual models have higher power** — prefer facelift examples for maximum output; hardware is identical to pre-facelift
- **Canadian SE is a bargain Type SH substitute** — if you want the look without the ATTS complexity, SE offers similar cosmetics at lower price
- **Australian H22A4 (1997–1998) is desirable** — lighter weight than USDM equivalents, H22A4 code (pre-H22Z1), ATTS-compatible
- **All 5th-gen Preludes are now 25-year exempt** — legal to import to the US from any market, increasing JDM/European variant availability
